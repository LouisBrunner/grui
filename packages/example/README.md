# godot-grui-example

A Godot 4 project demonstrating [`godot-grui`](https://crates.io/crates/godot-grui).

## What's inside

- [`basic.rs`](src/basic.rs) — a pause menu built with signals, event handlers, and custom components
- [`advanced.rs`](src/advanced.rs) — more advanced usage: generic components, `<For>`, `<Show>`, theming

## Running

1. Build the extension: `cargo build` from the workspace root
2. Open `packages/example` as a Godot 4 project
3. Run `test.tscn`
